# Desafio2 – Classificação de Imagens

Este projeto utiliza `TensorFlow` e outras bibliotecas para treino, avaliação e visualização de modelos de classificação de imagens (ex: `cats_and_dogs`).

--- Passo para correr os scripts:

# 1. Clonar o projeto 
Abrir uma janela de terminal e fazer:
cd Desafio2

# 2. Criar ambiente virtual
python3 -m venv venv

# 3. Ativar ambiente virtual
# macOS/Linux:
source venv/bin/activate
# Windows:
.\venv\Scripts\activate

# 4. Instalar dependências
pip install -r requirements.txt

# 5. Criar o Kernel
python -m ipykernel install --user --name=desafio2 --display-name "Python (Desafio2)"

# 5. Escolher o Kernel criado
Select Kernel -> Jupyter Kernel -> Python (Desafio2)
